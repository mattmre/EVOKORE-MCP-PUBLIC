Tips for first-time users

  • This wiki is generated. Re-run npm run wiki:build after pulling new skills or editing .env.example.
  • The skills index does not fetch live from mcp.config.json — it reflects the SKILL.md set at build time.
  • Some SKILL.md files are adapters of upstream submodules. They cite upstream, upstream-sha, and upstream-path in their frontmatter.
  • If your browser blocks file:// fetches, run python -m http.server in wiki/ and open http://localhost:8000.
  • Look at the presentations/ directory (if present in your checkout) for visual walkthroughs; this wiki is the deep reference.

What is EVOKORE-MCP?

EVOKORE-MCP is a multi-server MCP (Model Context Protocol) aggregator. It proxies child MCP servers, layers skill discovery on top, and adds session continuity, fleet orchestration, role-based access, and human-in-the-loop approval. See the project README.md for full setup.
