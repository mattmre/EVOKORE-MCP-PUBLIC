# ARCH-AEP Runner

This skill interacts with the ARCH-AEP (Agentic Engineering and Planning) workflows.

## Workflows

### 1. ARCH-AEP Review Cycle
- Review `CLAUDE.md` and `ARCH-AEP` information (e.g., `core/arch/README.md`, `core/arch/workflow.md`) to understand what the ARCH-AEP review cycle entails.
