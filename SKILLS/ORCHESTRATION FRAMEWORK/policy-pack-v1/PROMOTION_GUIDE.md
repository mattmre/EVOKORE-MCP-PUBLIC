# PROMOTION_GUIDE.md (Policy Pack v1)

Purpose: Rules for promoting reusable assets into shared skill libraries.

## Promote if
- Reusable across repos and models.
- Includes acceptance checks and evidence.
- Avoids repo-specific paths and secrets.

## Required Notes
- Why reusable?
- Acceptance checks and verification evidence.
- Security posture and risk triggers (if any).

## Log
- Update CHANGELOG with the promotion decision.
