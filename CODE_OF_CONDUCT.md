# Code of Conduct

This project adopts the **Contributor Covenant, version 2.1** as its Code of Conduct.

The full text is published and maintained upstream at:

<https://www.contributor-covenant.org/version/2/1/code_of_conduct.html>

All contributors, maintainers, reviewers, and participants in project spaces
(issues, pull requests, discussions, comments) are expected to read and abide
by it.

## Reporting

If you experience or witness behavior that violates this Code of Conduct,
report it privately to the project owner:

- **Project contact:** [@mattmre](https://github.com/mattmre) via
  [GitHub private vulnerability reporting](https://github.com/mattmre/EVOKORE-MCP-PUBLIC/security/advisories/new)
  (use this channel for confidential reports — it is the only private channel
  guaranteed to reach the maintainer).

Reports will be handled confidentially. The project owner reserves the right
to take any action deemed appropriate, up to and including a temporary or
permanent ban from project spaces.

## Why link-only

The full Contributor Covenant text is intentionally **not duplicated** in this
repository. The upstream source is the canonical version, is kept current by
the Contributor Covenant maintainers, and is reachable from the link above.
